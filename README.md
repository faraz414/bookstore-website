# Bookstore Website

A simple front-end web application built using HTML, CSS, and JavaScript.

## Features
- Multi-page layout (Home, About, Contact, Payment, Success)
- Styled using custom CSS
- Interactive behaviour implemented with JavaScript
- Image-based product display

## Project Structure
- `index.html` – main landing page
- `css/` – stylesheets
- `js/` – JavaScript functionality
- `images/` – image assets used in the site

## Technologies Used
- HTML
- CSS
- JavaScript
